Getting Started

After installing Marigold from the Shopify Theme Store, go to Online Store > Customize to start configuring your store.

Marigold ships with a bright, sunny palette — warm white backgrounds, golden accents, and bold uppercase headings. All colors, fonts, spacing, and layout options are fully customizable through the theme editor — no code required.

First Steps

  • Upload your logo in Header > Logo image
  • Set up your navigation menu in Header > Menu
  • Add a hero image to your homepage hero section
  • Select a featured collection to display on the homepage
  • Configure the announcement bar with your shipping offer
  • Set up footer blocks (link columns + newsletter signup)

Theme Settings

Access global theme settings via Customize > Theme settings (gear icon).

Colors

Marigold includes 7 customizable color tokens:

  • Background — Main page background (default: #FFFDF5 sunny white)
  • Foreground — Body text color (default: #2B2B3D soft black)
  • Accent — Buttons, links, highlights (default: #F5A623 golden marigold)
  • Border — Dividers and outlines (default: #F0E6D2 warm bisque)
  • Surface — Card and input backgrounds (default: #FFF5E1 light honey)
  • Sale — Sale price color (default: #C75D5D)
  • Sold out — Sold out badge color (default: #666666)

Typography

Choose any two fonts from the Shopify font library. Marigold defaults to:

  • Headings: Josefin Sans (geometric sans-serif)
  • Body: Work Sans (clean sans-serif)

The type scale uses fluid sizing with clamp() — text scales smoothly between mobile and desktop without breakpoints.

Layout

  • Max page width — Controls the maximum content width (default: 90rem)
  • Page margin — Minimum side padding (default: 24px)
  • Input corner radius — Rounds form inputs (default: 8px)
  • Button corner radius — Rounds buttons (default: 12px)

Sections

Marigold includes 22+ sections. All sections use the theme editor for configuration — no code editing needed.

Homepage Sections

  • Announcement bar — Dismissible top bar with custom colors. Great for shipping offers or promotions. Stays dismissed per session.
  • Hero — Full-width banner with image, overlay, heading, subheading, and up to 2 CTA buttons. Bold uppercase styling.
  • Featured collection — Product grid with playful card hover effects (lift + scale), golden shadows, and product badges.
  • Image with text — Two-column layout with image and rich text. Image position flips left/right.
  • Rich text — Centered text block with heading, body, and optional CTA button.
  • Video — YouTube or Vimeo embed with optional cover image and play button.
  • Testimonials — Quote cards in a responsive grid. Up to 6 testimonials.
  • Multi-column — Image + text columns for features or info blocks. Up to 6 columns.
  • Collapsible content — FAQ accordion using native <details> elements. No JavaScript. Up to 12 rows.
  • Logo list — Partner/press logos in a row. Grayscale with color on hover. Up to 8 logos.

Template Sections

  • Product — Two-column layout with image gallery and sticky product info. Variant swatches, quantity selector, dynamic checkout button, image zoom on hover. Supports Shopify-hosted and YouTube/Vimeo product videos. Stock counter shows "Only X left" when inventory is low. Sticky Add to Cart on mobile.
  • Collection — Product card grid with sort dropdown, storefront filters, pagination, and quick buy buttons on hover.
  • Cart drawer — Slide-out AJAX cart with quantity updates, order notes, no page reload.
  • Blog — Article cards with image, title, date, and excerpt.
  • Article — Full article with featured image, author, date, and rich text content.
  • Search — Search form with product result grid and pagination.
  • Password — Pre-launch page with shop branding, newsletter signup, and collapsible password entry.
  • Recommended products — "You may also like" section using Shopify's product recommendations API.
  • Contact form — Built-in contact form section.
  • Custom liquid — Inject custom Liquid code anywhere on the page.

Global Components

  • Sticky header — Frosted glass blur effect, stays visible on scroll.
  • Breadcrumbs — Automatic breadcrumb navigation on all pages except homepage.
  • Back-to-top button — Appears on scroll, smooth scroll to top.
  • Newsletter popup — Email capture popup with configurable delay.
  • Product badges — Automatic Sale and Sold Out badges on product cards.
  • Image rollover — Second product image reveals on hover in collection grids.
  • Quick buy — Add to cart directly from collection pages for single-variant products.
  • Stock counter — "Only X left in stock" when inventory is low.
  • Sticky Add to Cart — Buy button sticks to bottom on mobile screens.

Typography

Marigold uses a fluid type scale based on CSS clamp(). Sizes adjust automatically between mobile and desktop:

  • text-xs — Small labels, captions
  • text-sm — Secondary text, nav links
  • text-base — Body text
  • text-lg — Lead text, subtitles
  • text-xl to text-4xl — Headings

Marigold uses bold uppercase headings with wide letter-spacing for an energetic, confident feel.

Colors

Marigold ships with a sunny, optimistic palette — warm whites, golden accents, and honey-toned surfaces. Every color is customizable through theme settings.

Tip: For a softer look, try swapping the accent to #E05D6F (playful rose) for a warmer, more feminine feel.

RTL Support

Marigold fully supports right-to-left languages (Arabic, Hebrew, etc.). All layouts use CSS logical properties (margin-inline, padding-block, etc.) so the entire theme mirrors automatically when a RTL locale is selected.

No additional configuration is needed — Shopify handles the dir attribute automatically based on the selected language.

Performance

Marigold is built for speed:

  • No third-party JavaScript dependencies
  • Critical CSS loaded with preload
  • Lazy loading on all below-fold images
  • Fluid typography — no JavaScript resize handlers
  • Minimal DOM — no wrapper-heavy markup
  • CSS-only animations for card hovers and transitions

FAQ

Is Marigold only for baby stores?

No. Marigold is designed with baby and kids brands in mind, but its bright, joyful aesthetic works for any store that wants an energetic, optimistic feel — party supplies, craft kits, educational products, and more.

Can I tone down the brightness?

Yes. Every color is customizable. Swap the accent to a muted tone, reduce the background warmth, or even go dark — all components use CSS custom properties, so changes propagate instantly.

Does Marigold work with Shopify Markets?

Yes. All prices use the | money filter and all text uses translation keys, so multi-currency and multi-language setups work out of the box.

Is Marigold based on Dawn?

No. Marigold is built on the Skeleton base theme — original code, no Dawn dependencies, and full eligibility for the Shopify Theme Store.
