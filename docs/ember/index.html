Getting Started

After installing Ember from the Shopify Theme Store, go to Online Store > Customize to start configuring your store.

Ember ships with a dark color palette by default. All colors, fonts, spacing, and layout options are fully customizable through the theme editor — no code required.

First Steps

  • Upload your logo in Header > Logo image
  • Set up your navigation menu in Header > Menu
  • Add a hero image to your homepage hero section
  • Select a featured collection to display on the homepage
  • Configure footer blocks (link columns + newsletter signup)

Theme Settings

Access global theme settings via Customize > Theme settings (gear icon).

Colors

Ember includes 7 customizable color tokens:

  • Background — Main page background (default: #1A1A1A)
  • Foreground — Body text color (default: #F5F0EB)
  • Accent — Links, buttons, highlights (default: #C9A96E)
  • Border — Dividers and outlines (default: #2E2E2E)
  • Surface — Card and input backgrounds (default: #222222)
  • Sale — Sale price color (default: #C75D5D)
  • Sold out — Sold out badge color (default: #666666)

Typography

Choose any two fonts from the Shopify font library. Ember defaults to:

  • Headings: Playfair Display (serif)
  • Body: Work Sans (sans-serif)

The type scale uses fluid sizing with clamp() — text scales smoothly between mobile and desktop without breakpoints.

Layout

  • Max page width — Controls the maximum content width (default: 90rem)
  • Page margin — Minimum side padding (default: 24px)
  • Input corner radius — Rounds form inputs (default: 4px)
  • Button corner radius — Rounds buttons (default: 0px for sharp edges)

Sections

Ember includes 22 sections. All sections use the theme editor for configuration — no code editing needed.

Homepage Sections

  • Announcement bar — Dismissible top bar with custom colors. Limit 1. Stays dismissed per session.
  • Hero — Full-width cinematic banner with image, overlay, heading, subheading, text, and up to 2 CTA buttons. Supports left/center/right text alignment.
  • Featured collection — Asymmetric editorial product grid with hero tile. Select any collection.
  • Image with text — Two-column layout with image and rich text. Image position flips left/right.
  • Rich text — Centered text block with heading, body, and optional CTA button.
  • Video — YouTube or Vimeo embed with optional cover image and play button. Supports full-width mode.
  • Testimonials — Quote cards in a responsive grid (1/2/3 columns). Up to 6 testimonials.
  • Multi-column — Image + text columns for features or info blocks. Up to 6 columns.
  • Collapsible content — FAQ accordion using native <details> elements. No JavaScript. Up to 12 rows.
  • Logo list — Partner/press logos in a row. Grayscale with color on hover. Up to 8 logos.

Template Sections

  • Product — Two-column layout with image gallery and sticky product info. Variant swatches, quantity selector, dynamic checkout button. Shows vendor, handles sale/sold-out states.
  • Collection — Asymmetric product grid with sort dropdown, storefront filters (list + price range), and pagination.
  • Cart — Line items with AJAX quantity updates (no page reload). Responsive layout hides price columns on mobile.
  • Blog — Article cards with image, title, date, and excerpt.
  • Article — Full article with featured image, author, date, and rich text content.
  • Search — Search form with product result grid and pagination.
  • Password — Pre-launch page with shop branding, newsletter signup, and collapsible password entry.

Typography

Ember uses a fluid type scale based on CSS clamp(). Sizes adjust automatically between mobile and desktop:

  • text-xs — Small labels, captions
  • text-sm — Secondary text, nav links
  • text-base — Body text
  • text-lg — Lead text, subtitles
  • text-xl to text-4xl — Headings

Colors

Ember ships dark by default but is fully customizable. To create a light theme, simply swap the background and foreground colors in theme settings. All components use CSS custom properties, so changes propagate instantly.

Tip: For a light theme, try Background #FAFAF8, Foreground #1A1A1A, and adjust Surface/Border accordingly.

RTL Support

Ember fully supports right-to-left languages (Arabic, Hebrew, etc.). All layouts use CSS logical properties (margin-inline, padding-block, etc.) so the entire theme mirrors automatically when a RTL locale is selected.

No additional configuration is needed — Shopify handles the dir attribute automatically based on the selected language.

Performance

Ember is built for speed:

  • Zero external JavaScript dependencies
  • CSS-only scroll animations via animation-timeline: view() with graceful fallback
  • Critical CSS loaded with preload
  • Lazy loading on all below-fold images
  • Fluid typography — no JavaScript resize handlers
  • Minimal DOM — no wrapper-heavy markup

FAQ

Can I use Ember for a light-colored store?

Yes. Ember ships dark but every color is customizable. Swap background and foreground in theme settings for a light mode.

Does Ember work with Shopify Markets?

Yes. All prices use the | money filter and all text uses translation keys, so multi-currency and multi-language setups work out of the box.

Is Ember based on Dawn?

No. Ember is built on the Skeleton base theme. This means original code, no Dawn dependencies, and full eligibility for the Shopify Theme Store.
