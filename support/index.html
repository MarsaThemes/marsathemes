Support

Need help with your Marsa theme? We're here to assist. Check the documentation for your theme first:

Umber docs Burrow docs

What We Help With

  • Theme installation and setup
  • Configuring theme settings, sections, and blocks
  • Color, typography, and layout customization
  • Bug reports and troubleshooting
  • Guidance on using theme features (filters, variants, etc.)

What's Not Covered

  • Custom code modifications or third-party app integrations
  • Store setup, shipping, payments, or Shopify account issues (contact Shopify Support)
  • Design or development services beyond the theme's built-in features

Response Time

We respond within 24 hours on business days (Monday–Friday). Complex issues may take an additional day.

Get in Touch
